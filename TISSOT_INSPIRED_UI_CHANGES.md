# Tissot-inspired UrbanWear UI redesign

This update rebuilds the UrbanWear homepage and navigation with a cleaner luxury-commerce approach inspired by the structure of Tissot's homepage, while keeping all UrbanWear branding, data, product logic, cart, wishlist, checkout, admin, and backend features unchanged.

## UI changes
- Replaced the previous homepage with a luxury editorial landing page.
- Added a slim announcement bar.
- Added a restrained centered brand header.
- Added utility actions for search, account/login, wishlist, and bag.
- Added a desktop mega menu for Shop with Categories, Selections, and a feature panel.
- Added a refined mobile slide drawer with search, category links, selections, account actions, wishlist, bag, and tracking.
- Added large product-focused hero section.
- Added two large department cards for Men and Women.
- Added compact category tiles for Men, Women, Streetwear, and Accessories.
- Added product rail for new arrivals/featured products.
- Added editorial band and story cards.
- Added a minimal service row.
- Simplified footer to Shop and Help groups.

## Responsive changes
- Header adapts across desktop, tablet, and mobile.
- Desktop mega menu hides on smaller screens.
- Mobile drawer uses overlay and body scroll lock.
- Hero, category grids, product rail, story cards, and footer collapse cleanly on mobile.
