# Homepage and Menu Redesign

This update completely replaces the previous homepage and navigation/menu presentation with a new editorial fashion eCommerce layout.

## Homepage

- Replaced old homepage layout with a new editorial UrbanWear landing page.
- Added large fashion-first hero section with strong headline and simplified calls to action.
- Added product-led lookbook area using live product data.
- Added clean category edit cards for Men, Women, Streetwear, and Accessories.
- Added compact black capsule banner.
- Added new arrivals product grid with wishlist and add-to-bag actions.
- Removed old homepage clutter, long service/detail blocks, and repeated promotional sections.

## Menu and Navigation

- Replaced the old menu design with a new pill-shaped floating header.
- Added a redesigned left slide drawer menu.
- Added drawer overlay and close behavior.
- Added mobile-friendly search inside the drawer.
- Added simplified desktop navigation.
- Added responsive header actions for login, wishlist, and bag.
- Improved mobile layout so the menu, logo, and bag stay clean in one row.
